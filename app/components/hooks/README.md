# Hooks

Custom React hooks reutilizáveis.

## Arquivos a serem migrados:
- `useModuleAccess.jsx` - Hook para controle de acesso por módulo
- `useScale.jsx` - Hook para escala responsiva

## Exemplo de uso:
```jsx
import { useModuleAccess } from '@/hooks/useModuleAccess';
// ou
import { useModuleAccess } from '@/components/hooks';
```
