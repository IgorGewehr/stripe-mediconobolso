# Layout

Componentes de layout e navegação da aplicação.

## Arquivos a serem migrados:
- `sidebar.jsx` - Barra lateral de navegação
- `topAppBar.jsx` - Barra superior
- `bottomNavigation.jsx` - Navegação inferior mobile
- `helpCenter.jsx` - Centro de ajuda
- `ModuleProtection.jsx` - HOC de proteção por módulo
- `ProtectedRoute.jsx` - HOC de rota protegida

## Exemplo de uso:
```jsx
import { Sidebar } from '@/layout/sidebar';
// ou
import { Sidebar } from '@/components/layout';
```
