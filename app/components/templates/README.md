# Templates

Templates de páginas completas.

## Arquivos a serem migrados:
- `authTemplate.jsx` - Template de autenticação
- `dashboardTemplate.jsx` - Template do dashboard
- `doctorAITemplate.jsx` - Template do médico AI
- `pacienteTemplate.jsx` - Template da página de paciente
- `pacienteCadastroTemplate.jsx` - Template de cadastro de paciente
- `patientsListTemplate.jsx` - Template da lista de pacientes
- `prescriptionListTemplate.jsx` - Template da lista de receitas
- `receitasTemplate.jsx` - Template de receitas
- `userDataTemplate.jsx` - Template de dados do usuário
- `userProfileTemplate.jsx` - Template de perfil do usuário
- `centralAjudaTemplate.jsx` - Template da central de ajuda

## Exemplo de uso:
```jsx
import { DashboardTemplate } from '@/templates/DashboardTemplate';
// ou
import { DashboardTemplate } from '@/components/templates';
```
